## Prerequisites

* Node JS

## To run the Project

* Run `https://github.com/mohammadumar28/bigbinary-interview-for-mohammadumar28.git` to clone the project
* Go to the cloned directory
* Run `npm install` to install node modules
* Run `npm start` to start the app
